## Definitie

Byte Pair Encoding (BPE) is een data-gedreven tokenisatiemethode die start met losse tekens en herhaaldelijk de meest frequente aangrenzende symboolparen samenvoegt om een compact subwoord-vocabulaire voor taalmodellen te bouwen.
