## Definition

Ein Embedding-Modell ist ein [neuronales Netz](/de/glossary/neural-network/), das Eingaben – etwa Text, Code oder Bilder – in einen kontinuierlichen [Vektorraum](/de/glossary/embedding-space/) projiziert, in dem semantisch ähnliche Elemente nahe beieinander liegen.
