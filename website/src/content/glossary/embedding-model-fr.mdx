## Definition

Un modèle d’embedding est un [réseau de neurones](/fr/glossary/neural-network/) qui projette des entrées – texte, code ou images – dans un [espace vectoriel](/fr/glossary/embedding-space/) continu où les éléments sémantiquement proches sont situés à proximité.
