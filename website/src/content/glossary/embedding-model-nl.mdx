## Definition

Een embeddingmodel is een [neuraal netwerk](/nl/glossary/neural-network/) dat invoer – zoals tekst, code of afbeeldingen – projecteert naar een continue [vectorruimte](/nl/glossary/embedding-space/) waarin semantisch gelijkaardige items dicht bij elkaar liggen.
