## Définition

Un réseau feed-forward est une architecture de [réseau de neurones](/fr/glossary/neural-network/) dans laquelle les signaux se propagent uniquement vers l'avant, des entrées vers les couches cachées puis vers la sortie, sans boucles ni rétroactions.
