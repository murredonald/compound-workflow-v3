## Definition

Function Calling ist ein Muster, bei dem ein Sprachmodell eine strukturierte Ausgabe erzeugt, die angibt, welche externe Funktion oder welches Tool mit welchen Argumenten aufgerufen werden soll, um etwa APIs, Datenbanken oder andere Systeme einzubinden.
