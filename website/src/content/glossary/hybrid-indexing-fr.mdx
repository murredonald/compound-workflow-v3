## Definition

L’hybrid indexing stocke les documents dans plusieurs index – par exemple un index [BM25](/fr/glossary/bm25/) et un index vectoriel – afin que les requêtes puissent être évaluées à la fois sur les mots‑clés et sur la [similarité sémantique](/fr/glossary/semantic-similarity/).
