## Definition

Index Refresh umfasst das erneute Einbetten von Inhalten, den Wiederaufbau des Index oder das Hinzufügen neuer Segmente, damit Retrieval mit aktuellen Daten und Embedding‑Modellen im Einklang bleibt.
