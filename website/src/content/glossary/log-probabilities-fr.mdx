## Définition

Les log probabilities sont les logarithmes naturels des probabilités qu'un modèle de langage attribue aux différents tokens possibles suivants, souvent exposés via les API pour le débogage, la [calibration](/fr/glossary/calibration/) et des stratégies de décodage personnalisées.
