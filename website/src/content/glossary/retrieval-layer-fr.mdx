## Definition

Dans une architecture [RAG](/fr/glossary/rag/), la couche de récupération recherche dans un index et renvoie les documents ou passages les plus pertinents pour une [requête](/fr/glossary/prompt/) utilisateur ou une sous-requête générée par le modèle.
